# Communities_Sensitivity

* Python 2.7.9 (Anaconda 2.2.0)
* python-igraph 0.7.1.post4 (containing all community detection algorithms except Spectral clustering and Metis)
* scipy 0.15.1
* networkx 1.9.1
* conda 3.10.0
* metis 0.1a0
* scikit-learn 0.15.2
* matplotlib 1.4.3
* numpy 1.9.2
